# Intelligence-greenhouse

# 项目简介
构建多节点分布式嵌入式系统，实现小车智能巡检、温室环境数据采集、病虫害检测、自动化控制及数据上云。

# 项目结构
本项目包含以下几个主要部分：
- **安卓应用**：移动端控制 / 显示界面
- **开发板代码**：开发板相关代码
- **ESP32 代码**：基于 ESP32 芯片的固件和应用
- **STM32 代码**：基于 STM32 系列单片机的程序
